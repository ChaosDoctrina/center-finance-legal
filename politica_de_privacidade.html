Política de Privacidade

Última atualização: 23/12/2025

1. Coleta de Dados

Coletamos apenas os dados necessários para o funcionamento do sistema:

  • Nome completo
  • E-mail
  • Senha (armazenada de forma criptografada)
  • Número de telefone

2. Dados Financeiros

O usuário pode inserir dados financeiros manualmente ou via mensagens enviadas pelo WhatsApp.

Esses dados não representam informações bancárias oficiais, sendo apenas registros pessoais.

3. Uso de Inteligência Artificial

Mensagens de texto ou áudio enviadas via WhatsApp podem ser processadas por serviços de inteligência artificial (OpenAI) exclusivamente para transformar o conteúdo em dados organizados no sistema.

Nenhum dado é utilizado para treinamento público de modelos.

4. Integrações com Terceiros

Utilizamos serviços de terceiros para operação da plataforma:

  • WhatsApp (Meta) – recepção de mensagens
  • OpenAI – processamento de linguagem
  • Stripe – processamento de pagamentos

Cada serviço possui sua própria política de privacidade.

5. Pagamentos

Os dados de pagamento são processados exclusivamente pela Stripe. O Center Finance não armazena dados completos de cartão de crédito.

6. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros, exceto quando necessário para funcionamento do serviço ou exigência legal.

7. Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados, incluindo:

  • Criptografia
  • Controle de acesso
  • Ambiente seguro

8. Direitos do Usuário (LGPD)

O usuário pode solicitar:

  • Acesso aos seus dados
  • Correção
  • Exclusão da conta

9. Retenção de Dados

Os dados são mantidos enquanto a conta estiver ativa ou conforme exigido por lei.

10. Alterações

Esta política pode ser atualizada a qualquer momento. A versão mais recente estará sempre disponível no site.
